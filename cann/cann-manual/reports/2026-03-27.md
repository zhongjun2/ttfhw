| P1-02 | P1（绕路可过） | 官方 AscendHub 镜像地址 `ascendhub.huawei.com` DNS 无法解析（网络限制，该域名在当前环境不可访问） | 无法使用文档指定的镜像；使用公开 Docker Hub 镜像 `ascendai/cann:latest` 作为绕路方案 |

- P2-04：使用 `sg docker -c "..."` 临时切换 docker 组权限，正常执行所有 docker 命令。
