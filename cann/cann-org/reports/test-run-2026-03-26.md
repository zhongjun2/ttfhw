## 九、异常说明

1. **SPA 文档（了解阶段）：** hiascend.com 官方文档为 Nuxt.js 单页应用，自动化测试无法抓取正文内容，需要使用浏览器或搜索摘要替代。建议 hiascend.com 增加静态文档版本或 API 接口。

2. **ascendhub.huawei.com 不可达：** 该域名无公共 DNS 记录，仅华为内网可访问。官方文档中的主要镜像地址对外部开发者不可用，影响 TTFHW 体验。建议文档中优先标注 SWR 或 quay.io 地址。

3. **SWR 需要认证：** `swr.cn-south-1.myhuaweicloud.com` 虽然 DNS 可达，但需要华为云账号才能拉取，增加了新开发者的使用门槛。

4. **docker 组权限：** 新安装的机器默认不会将用户加入 docker 组，需要 sudo 操作。这是常见的一次性配置，文档应明确提示。

5. **quay.io 镜像版本超前：** quay.io 提供的是 CANN 9.0.0-beta.1（beta 版），而非文档中的 8.0.RC3。在需要精确版本匹配的场景下可能有兼容性风险。
