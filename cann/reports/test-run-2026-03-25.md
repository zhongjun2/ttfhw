## 异常记录

### 环境问题

| # | 问题 | 受影响阶段 | 解决方法 |
|---|------|-----------|---------|
| E1 | `BING_API_KEY` 未配置，Bing API 返回 401 | 了解 | 在 [Azure 门户](https://portal.azure.com/) 创建 Bing Search v7 资源，获取 API Key |
| E2 | `docker` 用户组权限不足，`/var/run/docker.sock` 拒绝访问 | 获取、使用 | `sudo usermod -aG docker $USER && newgrp docker` |
| E3 | `ascendhub.huawei.com` DNS 解析失败，`docker pull` 无法执行 | 获取、使用 | 需在华为云或已授权企业网络中运行；另需先在 [id.hiascend.com](https://id.hiascend.com) 注册账号并执行 `docker login ascendhub.huawei.com` |
| E4 | `GITCODE_TOKEN` 未配置 | 贡献 | 在 [Gitcode 个人访问令牌页](https://gitcode.com/-/profile/personal_access_tokens) 创建 PAT，勾选 `api` 权限 |
| E5 | `ttfhw-bot/cann` fork 仓库不存在 | 贡献-PR | 在 Gitcode 创建 `ttfhw-bot` 账号，fork [gitcode.com/ascend/cann](https://gitcode.com/ascend/cann) |

| # | Bug | 严重度 | 说明 |
